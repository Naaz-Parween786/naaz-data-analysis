# 📊 Data Analysis App

A simple and interactive Data Analysis web app built with Python and Streamlit.
Upload any CSV file and instantly explore your data — no coding required!

---

## 🚀 Live Demo
(https://naaz-data-analysis-at3hnp4ezhzjsywrts3deu.streamlit.app/)

---

## ✨ Features

- 📁 Upload any CSV file instantly
- 📋 View dataset overview — total rows, columns, missing values, duplicate rows
- 👀 Preview first 5 rows of your data
- 📊 View statistics — count, mean, min, max, standard deviation for all numeric columns
- 📈 Visualize data with interactive bar charts
- 📥 Download your data as CSV file

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| Python | Main programming language |
| Streamlit | Web app framework |
| Pandas | Data loading and analysis |
